# FSKTM PSM Management System

This is a web-based system designed for managing Final Year Project (Projek Sarjana Muda) processes for FSKTM students and supervisors at UTHM.

## Features
- Student login & supervisor login
- Upload final year project chapters
- Supervisor review & feedback
- Notification system for submissions and approvals
- Dashboard for both students and supervisors
- Session & role-based access control

## Technology Stack
- **Frontend:** HTML, CSS, Bootstrap
- **Backend:** PHP (procedural & prepared statements)
- **Database:** MySQL
- **Server:** Apache (XAMPP)
